# 3d-Printed-5DOF-robotic-arm-Isaac-Simulator-integration-inverse-kinematics
3D Printed 5-DOF Robotic Arm with Isaac Sim &amp; Inverse Kinematics  A 3D-printed 5-DOF robotic manipulator designed, built, and simulated from scratch. The project combines mechanical design, inverse kinematics, and digital-twin simulation using NVIDIA Isaac Sim.
